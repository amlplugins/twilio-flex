# @amlplugins/twilio-flex

Native `twilio` SDK re-exported under the `@amlplugins` namespace.

> Twilio Flex — programmable cloud contact center platform with channels, workflows, and analytics.

## Install

```bash
npm install @amlplugins/twilio-flex
```

## Use

```ts
import { /* named exports from twilio */ } from "@amlplugins/twilio-flex";
// or
import sdk from "@amlplugins/twilio-flex";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
